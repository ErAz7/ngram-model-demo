# N-Gram Model Demo [![npm-shield]][npm] ![ts-shield]
> A demo showcasing the usage of [ngram-model][npm] npm package

## Run locally

Run 
```bash
yarn
``` 
to install, then 
```bash
yarn dev
``` 
to run development server

## Build

Run 
```bash
yarn build
```
